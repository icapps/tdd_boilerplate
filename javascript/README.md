# TDD javascript boilerplate

Simple package that can be used as a boilerplate for practicing TDD using Javascript.

## Features

* Jest: https://jestjs.io/docs/en/getting-started.html
* ES6 (via Babel)

## Commands

### Init repo

```bash
$ yarn install
```

### Run single test 

```bash
$ yarn run test
```

### Run tests on file change(s)

```bash
$ yarn run watch
```
